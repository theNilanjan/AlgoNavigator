# AlgoNavigator 🚀

An interactive Pathfinding and Graph Traversal Visualizer built using **C++** and **SFML**.

## Overview

AlgoNavigator is a visualization tool designed to help users understand how pathfinding algorithms work on a grid-based environment. The project provides a graphical interface where users can create obstacles, define start and end points, and watch algorithms explore the grid in real time.

The primary goal of this project is to make learning graph algorithms intuitive and engaging through visual representation.

## Current Features

* Grid-based visualization
* Interactive wall placement
* Start node selection
* End node selection
* Breadth-First Search (BFS) visualization
* Real-time algorithm execution

## Technologies Used

* C++
* SFML 3.x
* Object-Oriented Design
* Graph Algorithms

## Controls

| Key         | Action            |
| ----------- | ----------------- |
| S           | Select Start Node |
| E           | Select End Node   |
| B           | Run BFS Algorithm |
| Mouse Click | Place Nodes/Walls |
